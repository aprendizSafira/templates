O que é Templates?
Em programação é código html misturado com algumas palavras-chaves.E depois o php vai transforma 
essa chave no seu valor real.
Exemplo:
<title>{titulo}</title>//Depois essa chave vai ser transformada no titulo real;

Assim não precisa colocar código php no dentro do html. Meio para organizar as linguagens dentro
de cada arquivo.
Existem várias Bibliotecas para manipular o template. Mas como estou aprendendo a programar, vou 
fazer o próprio sistema inicialmente.
